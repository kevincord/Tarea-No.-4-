# Principios DRY, KISS y YAGNI

Este repositorio contiene ejemplos de los principios de desarrollo de software:

## DRY (Don't Repeat Yourself)
Ejemplo en JavaScript donde se elimina código repetido.

## KISS (Keep It Simple, Stupid)
Ejemplo en Java donde se simplifica el código.

## YAGNI (You Aren't Gonna Need It)
Ejemplo en C# donde se elimina código innecesario.

Cada ejemplo muestra el código antes y después aplicando el principio.
